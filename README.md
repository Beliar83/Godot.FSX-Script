# Godot.FSX-Script
Adding FSX as a scripting language to godot

Note: Currently this will either need the .NET Release of at least 4.4 dev 7, or a custom build with at least [PR 98545](https://github.com/godotengine/godot/pull/98545) and with [mono](https://github.com/godotengine/godot/tree/master/modules/mono#readme) enabled
